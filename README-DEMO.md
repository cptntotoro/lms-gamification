## Тестирование демо-эндпоинтов

Демо-эндпоинты предназначены для быстрой проверки UI-страниц.  
Все данные берутся из скрипта `data.sql` (12 студентов, 3 курса, 5 групп, 5 типов событий).

**Базовый URL:** `http://localhost:8080`

Предполагается, что приложение запущено и данные из `data.sql` загружены.

### 1. Виджет прогресса пользователя

**Путь:** `/api/v1/demo/users/{userId}`  
**Шаблон:** `widget.html`

| Сценарий                    | URL                             | Ожидаемый результат                                  |
|-----------------------------|---------------------------------|------------------------------------------------------|
| Виджет лидера (student007)  | `/api/v1/demo/users/student007` | Уровень 7, 1150 очков, прогресс до уровня 8          |
| Средний студент             | `/api/v1/demo/users/student001` | Уровень 5, 850 очков                                 |
| Несуществующий пользователь | `/api/v1/demo/users/student999` | Сообщение об ошибке "Данные пользователя недоступны" |

### 2. Полная статистика по курсу и группе (виджет + данные по курсу)

**Путь:** `/api/v1/demo/users/users/{userId}`  
**Обязательный параметр:** `courseId` (query)  
**Опциональный:** `groupId`

| Сценарий                          | URL                                                                    | Ожидаемый результат                                           |
|-----------------------------------|------------------------------------------------------------------------|---------------------------------------------------------------|
| Студент в MATH-101, группа M-21-1 | `/api/v1/demo/users/users/student001?courseId=MATH-101&groupId=M-21-1` | Очки по курсу 1240, место в группе, глобальный прогресс       |
| Лидер MATH-101 (M-21-2)           | `/api/v1/demo/users/users/student007?courseId=MATH-101&groupId=M-21-2` | Очки по курсу 1620 (лидер группы), высокий глобальный уровень |
| Лидер PROG-202                    | `/api/v1/demo/users/users/student012?courseId=PROG-202&groupId=P-22-2` | Очки по курсу 1180 (лидер курса), место в группе              |
| Весь курс MATH-101 (без группы)   | `/api/v1/demo/users/users/student007?courseId=MATH-101`                | Статистика по всему курсу MATH-101                            |
| Несуществующий курс               | `/api/v1/demo/users/users/student001?courseId=WRONG-COURSE`            | Ошибка "Курс не найден" или "Зачисление не найдено"           |
| Пользователь не зачислен на курс  | `/api/v1/demo/users/users/student001?courseId=HIST-303`                | Ошибка "Пользователь не зачислен на курс"                     |
| Без courseId                      | `/api/v1/demo/users/users/student001`                                  | Ошибка валидации "courseId обязателен"                        |

### 3. Лидерборд (персонализированный)

**Путь:** `/api/demo/leaderboard/course/{courseId}/...`  
**Шаблон:** `leaderboard.html`

| Сценарий                              | URL                                                                        | Ожидаемый результат                                   |
|---------------------------------------|----------------------------------------------------------------------------|-------------------------------------------------------|
| Лидерборд курса MATH-101 (все группы) | `/api/demo/leaderboard/course/MATH-101/user/student007`                    | Топ студентов курса + место и очки student007 (лидер) |
| Лидерборд группы M-21-1               | `/api/demo/leaderboard/course/MATH-101/groups/M-21-1/user/student001`      | Топ группы M-21-1 + место student001                  |
| Лидерборд курса PROG-202              | `/api/demo/leaderboard/course/PROG-202/user/student009`                    | Топ по PROG-202 + место student009                    |
| Пагинация (2-я страница)              | `/api/demo/leaderboard/course/MATH-101/user/student007?page=1&size=10`     | Вторая страница лидерборда курса MATH-101             |
| Максимальная страница                 | `/api/demo/leaderboard/course/MATH-101/user/student007?page=0&size=100`    | До 100 записей на странице                            |
| Несуществующий курс                   | `/api/demo/leaderboard/course/WRONG-COURSE/user/student007`                | Ошибка 404 или сообщение "Курс не найден"             |
| Несуществующая группа                 | `/api/demo/leaderboard/course/MATH-101/groups/WRONG-GROUP/user/student001` | Ошибка или пустой лидерборд                           |

### 4. Админ-панель (UserAdminPageController)

| Сценарий                    | URL                                | Ожидаемый результат                                   |
|-----------------------------|------------------------------------|-------------------------------------------------------|
| Главная админ-панель        | `/api/demo/admin`                  | Страница `admin` (дашборд или меню)                   |
| Профиль лидера              | `/api/demo/admin/users/student007` | Детальная информация: уровень 7, 1150 очков, прогресс |
| Профиль среднего студента   | `/api/demo/admin/users/student001` | Уровень 5, 850 очков                                  |
| Несуществующий пользователь | `/api/demo/admin/users/student999` | Сообщение "Пользователь с ID 'student999' не найден"  |

### 5. Страница типов событий (EventTypeAdminPageController)

| Сценарий                           | URL                                                | Ожидаемый результат                                                             |
|------------------------------------|----------------------------------------------------|---------------------------------------------------------------------------------|
| Список всех типов событий          | `/api/demo/admin/event-types`                      | Таблица с quiz (50), homework (100), attendance (20), project (300), bonus (80) |
| Вторая страница (если типов много) | `/api/demo/admin/event-types?page=1&size=3`        | Вторая страница списка типов событий                                            |
| Сортировка по названию             | `/api/demo/admin/event-types?sort=displayName,asc` | Список отсортированный по displayName                                           |

### 6. Список всех курсов пользователя + глобальный прогресс (UserCoursesController)

**Путь:** `/api/v1/leaderboard/users/{userId}/courses`  
**Метод:** `GET`

Возвращает глобальную статистику пользователя (уровень, общие очки, прогресс до следующего уровня) + полный список всех
курсов, на которые он зачислен.

| Сценарий                                   | URL                                            | Ожидаемый результат                                                                  |
|--------------------------------------------|------------------------------------------------|--------------------------------------------------------------------------------------|
| Пользователь с несколькими курсами (лидер) | `/api/v1/leaderboard/users/student007/courses` | Уровень 7, 1150 очков + **2 курса** (MATH-101: 1620 очков, группа M-21-2 + HIST-303) |
| Студент только MATH-101                    | `/api/v1/leaderboard/users/student001/courses` | 1 курс: MATH-101 (1240 очков, группа M-21-1), `totalCourses = 1`                     |
| Студент PROG-202 + HIST-303                | `/api/v1/leaderboard/users/student012/courses` | 2 курса: PROG-202 (1180 очков) + HIST-303 (640 очков)                                |
| Только один курс (PROG-202)                | `/api/v1/leaderboard/users/student009/courses` | 1 курс: PROG-202 (670 очков)                                                         |
| Студент с HIST-303                         | `/api/v1/leaderboard/users/student011/courses` | 2 курса (PROG-202 + HIST-303)                                                        |
| Несуществующий пользователь                | `/api/v1/leaderboard/users/student999/courses` | Ошибка "Пользователь не найден"                                                      |

**Примечание:**  
Курсы отсортированы по дате зачисления (новые сверху). В каждом элементе списка доступны `displayName`,
`totalPointsInCourse`, `groupId`, `enrolledAt`, `completedAt`.

### Общие рекомендации по тестированию

- **Браузер**: Ctrl + F5 / Ctrl + Shift + R — очистка кэша
- **Логи**: смотри консоль — там `log.debug` и `log.info` из контроллеров

Все сценарии используют реальные данные из `data.sql` — лидеры, группы и очки будут соответствовать вставкам.
